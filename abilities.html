The Finished Product

Browser Compatibility

Typically the websites I build are for the most modern version of each browser. This gives you a beautiful interface, animation and user experience that is simply cannot be efficiently incorporated when building a site to work with every browser.
With that said, I can absolutely build a beautiful interface for any browser going as far back as IE6.

Responsive

This means that the websites I build are designed to respond to each device immediately. This means you get one website that can be viewed in many different environments, without the need to create a new website for each environment.

Animations

These range all the way from building icons and elements that change visually based on a specific input to designing an actual animation with multiple elements moving in tandem with each other.

*Graphic Design

I build any number of icons to fit the theme of a site. These can be either regular images or a special type called SVG.These are retina ready/lossless, responsive, and incredibly small (file size).

Specialty

  • Hand Coded

    Every website I make is 100% guaranteed custom built from scratch to specifically fit your needs, never from a template.
    With that said, if you have a template that you would like to incorporate we can definitely do that.

  • Responsive

    You get a custom responsive layout meaning which means your layout is built specifically for your content. Which provides your users lossless access to content regardless of the device they use.

  • Ui / Ux

    Many web designers like to use these two interchangeably but they most certainly are not. Simply put: UI is the way it looks, and UX is how it feels. A site built from scratch means I can do things other designer simply cannot do.

  • Animations

    When I say animations it is mostly in the scope of web design, to enhance the UX (user experience). I can build complex animations but generally speaking minimalist animations resonate more with users.
