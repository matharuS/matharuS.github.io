Technical Specifications

TL;DR

Everything is custom built from scratch. By decreasing HTTP requests and only incorporating needed code your website is faster for users and costs less to host.

Responsive

I build Responsive webistes with two planes of logic. Smarter Design and Custom Break-points. Buy building and position the elements in the right way using the right styles and attributes, I can make your website respond naturally to the browser without including complex and unnessesary code. Then in combination with custom break-points relative to the content, your webiste is literally designed for every device.

Cross Browser

Generally speaking the websites i build are for the most modern version of each browser. It really isnt efficiently possible to create beautiful modern websites in browsers that werent designed for this. If your out on the highway driving a Lamborghini would you want to take the off ramp to an old and unmaintaned dirt path? With that said I can most certainly built you a beautiful site going as far back as IE8, it just wont be as fancy.

Animations

Built independintly or in combination with SVG, CSS3, Javascript and Velocity.js The animations implemented are completely dependent on the browsers you want your site to be accessable on The rule of thumb is: the older the broswer the lower the complexity

Icons/graphics

I can provide 3 types of icons: vector, regular(png/jpg/gif), or SVG Vector Graphics are great because they Regular SVG are undoubtably the best. Built in code so there are no http requests, literally infinite lossless scalability (retina ready) and they are incredibly small (~5kb svg VS +50kb regular)
